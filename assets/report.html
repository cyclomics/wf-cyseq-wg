General run statistics
Read yield evolution
This section provides an overview of the sequencing run, including the number of reads generated, total bases sequenced, and other key metrics.
The Sankey plot shows the flow of sequenced reads through consensus generation. It displays reads that pass or fail consenus. Reads that pass consensus are further categorized by their type (1D, 2D or 3D). Reads that fail consensus are further categorized by their reason for failure.
This section shows the yield of consensus read bases generated.
The Sankey plot shows the proportion of consensus bases by read type (1D, 2D, 3D) after successful consensus. When consensus fails, the underlying reason is displayed.
This section shows the yield of raw read bases analysed.
The Sankey plot shows the proportion of raw bases by read type (1D, 2D, 3D) after successful consensus. When consensus fails, the underlying reason is displayed.
